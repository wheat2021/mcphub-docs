## 图片（Image）

### 使用 Markdown

使用标准 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#images) 引入图片：

```md
![标题](/path/image.jpg)
```

注意：单个图片文件建议小于 5MB；更大的资源推荐托管在 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等平台，并以 URL 方式嵌入。

### 使用嵌入（Embeds）

若需更灵活的控制，可使用 HTML/Embed 方式（或配合 Mintlify 组件）：

```html
<img height="200" src="/path/image.jpg" />
```

### iFrame 示例

在文档中加载另一个 HTML 页面（常用于嵌入视频等）：

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
